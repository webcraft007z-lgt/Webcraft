#  Webcraft – Frontend Developer Portfolio

[![Live Demo](https://img.shields.io/badge/demo-online-00FFAA?style=flat-square)](https://your-deployed-url.netlify.app)
[![Built with](https://img.shields.io/badge/built%20with-HTML%2FCSS%2FJS-001F3D?style=flat-square)]()

A cutting‑edge, fully responsive portfolio website for a modern frontend developer.  
Designed with a **dark glassmorphic aesthetic**, an **infinite looping project belt**, and a fully functional contact form that calculates service prices including logo design.

##  Features

- **Infinite horizontal project belt** – auto‑scrolls with pause‑on‑hover, shows three live projects (The Daily Grind, NFT Horizon, Analytics CMS)
- **Fully responsive** – works on desktop, tablet, and mobile (hamburger menu on small screens)
- **Floating glass navbar** – beveled edges, semi‑transparent, with particle effects on hover/click
- **Dynamic contact form** – integrated with Web3Forms; users select a service (design only, design+dev, dev only) and optional logo design → price updates in real time
- **Custom SVG favicon & logo** – stylised “W” with neon mint accent
- **Smooth scroll animations** – powered by GSAP & ScrollTrigger
- **Custom scrollbar** – matches brand colours (Midnight + Spring Green)
- **No external backend needed** – static HTML/CSS/JS, works offline, easy to deploy
